# Tienda Online Demo

Este es mi proyecto donde hice una tienda en línea parecida a Mercado Libre. El objetivo fue mostrar productos, filtrarlos y poder agregarlos al carrito usando HTML, CSS y JavaScript. El diseño es sencillo pero funcional.

## Características del proyecto
- Muestra productos desde un arreglo en JavaScript.
- Sistema de búsqueda para encontrar productos por nombre.
- Filtro por categorías.
- Botón para agregar artículos al carrito.
- Interfaz parecida a Mercado Libre (colores, tarjetas, etc.).

## Tecnologías utilizadas
- **HTML** → estructura de la página.
- **CSS** → estilos y diseño.
- **JavaScript** → funciones, filtros y carrito.

## Archivos del proyecto
- `index.html` → Página principal.
- `style.css` → Estilos.
- `main.js` → Lógica de los productos y carrito.

## Buenas prácticas aplicadas
- Usé **nombres claros** en las variables.
- Separé el código en **HTML, CSS y JS** para que no esté todo mezclado.
- Intenté hacer el código lo más **ordenado** y con espacios.
- Utilicé **funciones reutilizables** para no repetir código.
- Comenté partes importantes para entender qué hace cada cosa.

## Nota final
Hice este proyecto como práctica y para aprender más sobre cómo funcionan las tiendas en línea con JavaScript básico.
